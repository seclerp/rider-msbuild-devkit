## [Unreleased]

## [1.2.1] - 2025-02-26

### Added

- Support for Rider 2025.1 EAP 5

## [1.2.0] - 2024-12-21

- Support for Rider 2024.3.2

## [1.1.1] - 2024-04-06

### Added

- Support for Rider 2024.1 RC1

### Added

- Run/Debug MSBuild tasks using new Run Configuration
- "MSBuild Library" C# project template
